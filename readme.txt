Instructions for the COM port: 

You can find two files uder /RxTxComm/x64 directory for x64 architecture systems and under /RxTxComm/x86 for x86 architecture systems: 

RXTXcomm.jar - must be copied into lib\ext\ directory of the active JRE (example: "C:\Program Files\Java\jdk1.8.0_65\jre\lib\ext\RXTXcomm.jar")   
rxtxSerial.dll - must be copied into bin\ directory of the active JRE (example: "C:\Program Files\Java\jdk1.8.0_65\jre\bin\rxtxSerial.dll")
